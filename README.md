# Nicanor Molinare — La Casa de Huéspedes

Un juego de terror psicológico en formato aventura de texto, jugable directamente en el navegador.

## Cómo jugar

1. Descargá `nicanor_molinare.html`
2. Abrilo en cualquier navegador (Chrome, Firefox, Edge)
3. Escribí tu nombre y comenzá a explorar

## Sobre el juego

- **Género:** Terror psicológico / Aventura de texto
- **Formato:** Archivo HTML único (~157 KB), sin dependencias externas
- **Original:** Portado desde Quest 5.8 (.aslx) a HTML/JS puro
- **Habitaciones:** 27 locaciones interconectadas
- **NPCs:** 5 personajes con diálogos reactivos al nivel de horror
- **Finales:** 5 finales diferentes según tus decisiones
- **Secretos:** 25 easter eggs ocultos que desbloquean el final verdadero

## Características

- Sistema de horror progresivo (0-20) que altera descripciones y efectos visuales
- Interfaz con sidebar (mapa, inventario, acciones contextuales, objetos en la habitación)
- Audio ambiental generado con Web Audio API
- Efectos CSS de horror (shake, vignette, pulso)
- Responsive para móvil
- Sin necesidad de servidor — funciona offline

## Comandos

`ir norte/sur/este/oeste` · `examinar [objeto]` · `hablar con [persona]` · `tomar [objeto]` · `usar [objeto]` · `inventario` · `ayuda`
